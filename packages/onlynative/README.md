# onlynative

Shorthand package for [`@onlynative/cli`](https://github.com/onlynative/ui/tree/main/packages/cli).

```bash
npx onlynative create my-app
npx onlynative init
npx onlynative add button card
```

This package delegates all commands to `@onlynative/cli`. See the [CLI docs](https://onlynative.github.io/ui/cli) for the full command reference.
